# Rainfall-Prediction
This is a Kaggle Competition project written in Python. 
We predicts whether there is rainfall at a location, based on (processed) infrared satellite image information.

# Data size 
1 million.

# Leanring method: 
1. Random Forest 
2. AdaBoost With Tree 
3. Gradient Boosting 

# Performance 
AUC : 0.78371 ;
